# NayePankh Foundation Website Redesign

A modern, responsive redesign of the NayePankh Foundation website built as a frontend development project. The goal of this project was to improve the visual design, user experience, responsiveness, and overall presentation of the foundation's mission and initiatives.

## ✨ Features

* Fully responsive design for mobile, tablet, and desktop devices
* Modern and clean NGO-inspired user interface
* Smooth scrolling navigation
* Mobile hamburger menu
* Dynamic navbar effects on scroll
* Interactive cards and hover effects
* Gallery and media coverage sections
* Contact form section
* Social media integration
* Optimized spacing, typography, and color system

## 🛠️ Tech Stack

* HTML5
* Tailwind CSS
* JavaScript (ES6)
* Font Awesome
* Netlify (Deployment)

## 📖 Website Sections

* Hero Section
* Impact Statistics
* Our Initiatives
* About Us
* Gallery
* Media Coverage
* Get Involved
* Contact Section
* Footer

## 🎯 Project Objectives

* Practice building complete, real-world landing pages
* Improve responsive design skills
* Apply JavaScript DOM manipulation and event handling
* Learn deployment and project presentation workflows
* Create a professional frontend project based on an existing NGO website

## 💡 Learning Highlights

During this project, I explored:

* Responsive layouts using Tailwind CSS
* Flexbox and Grid systems
* Component-based section design
* DOM manipulation and event listeners
* Website deployment using Netlify
* Iterative design improvements and rapid prototyping techniques
